# 包

import 的一般格式如下:

import "包的路径 或 URL地址"
1

当使用 . 作为包的别名时，可以不通过包名来使用其中的项目

当使用 _ 作为包的别名时，只导入其副作用，也就是说，只执行它的 init() 函数并初始化其中的全局变量
