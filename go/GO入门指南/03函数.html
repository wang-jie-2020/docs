# 函数

Go 里面有三种类型的函数：

  • 普通的带有名字的函数
  • 匿名函数或者lambda函数（参考 第 6.8 节 (opens new window)）
  • 方法（Methods，参考 第 10.6 节 (opens new window)）

Go不支持函数重载,不支持泛型.

Go默认使用值传参

# init

init 函数 在main 函数前自动执行，有些类似于构造函数

# defer

关键字 defer 的用法类似于面向对象编程语言 Java 和 C# 的 finally 语句块，它一般用于释放某些已分配的资源。只是类似,实际上执行时刻是不一样的。
