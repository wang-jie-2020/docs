## WebStorm 中跳转问题

在项目根目录下[文件：jsconfig.json]

```js
{
  "compilerOptions": {
    "baseUrl": "./",
    "paths": {
      "@/*": [
        "src/*"
      ]
    }
  },
  "exclude": [
    "node_modules",
    "dist"
  ]
}
```

   在div中增加些东西,或者通过css撑开
