通过Thread指定一个新的线程:

  • extends Thread

通过匿名或者Lambda表达式指定Thread任务:

new Thread(new Runnable() {
    @Override
    public void run() {
        try {
            Thread.sleep(1000);
        } catch (InterruptedException e) {
            throw new RuntimeException(e);
        }
        System.out.println("runnable method");
    }
}).start();
